# Aurora Sleep Technologies - Tracking Implementation Project

This is a professional landing page for a fictional bio-hacking hardware company. The project is designed to demonstrate advanced **Google Tag Manager (GTM)** and **Google Analytics 4 (GA4)** implementations.

## Key Features
- **Flowchart UI:** Explains the 3-stage sleep optimization methodology.
- **Investor Portal:** Interactive modal to track custom DataLayer events.
- **Sleep Audit Form:** Includes file attachment capability for lead generation tracking.
- **Video Integration:** Embedded YouTube content for engagement analysis.

## Tracking Tasks Implemented
1. **Form Submission:** Tracking lead generation and file uploads.
2. **Scroll Depth:** Measuring user interest in the scientific methodology.
3. **Link Clicks:** Tracking "Investor Relations" and social media clicks.
4. **Video Engagement:** Tracking starts and progress on the "Importance of Sleep" video.
5. **File Downloads:** Tracking clicks on the Clinical Whitepaper PDF.
